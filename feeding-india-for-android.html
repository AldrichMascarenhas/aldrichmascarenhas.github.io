Feeding India

Feeding India is an Android App built for FeedingIndia.org a youth run not for profit organization working to end Hunger and Malnutrition.

The prototype for the App was built at Techyon 2015, where we won 1st Place at the Hackathon. Aaron Colaco was resposible for building the Sinatra API, source for which can be found here.
