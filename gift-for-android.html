GIFt - Link a GIF

GIFt is an Android App that enabled users to share and view Gifs inline in Messaging Platforms that didn't have native Gif support.

GIFt uses the GIPHY Api that makes it dead simple for developers to incorporate GIPHY's vast library right inside of their apps. Retrofit and OkHttp are used to handle and make all Network Requests.

GIFt makes use of DeepLinkDispatch developed by AirBnB that provides a declarative, annotation-based API to define application deep links. DeepLinks are registered in the Activity of the App, and depending on the type of ID pulled out from the deep link, the appropriate GIF is downloaded and displayed.
